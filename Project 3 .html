THE GREAT ESCAPE

Game project for an experimental touch wall created by OiOi

Released 2021 Mac (Unity)

Game is playable at Xamk Kouvola Campus - Paja downstairs






Project

  • Game about an ongoing heist, where player has to collect as much loot during the escape to score as much as they can.
  • Theme of the game is derived from a farce comedy a “The Comedy of a Bank Robbery” in the Kouvola theatre 60th anniversary repertoire.
  • Developed by The South-Eastern University of Applied Sciences, XAMK
  • Commissioned by C3E www.xamk.fi/c3e
  • Technical support by OiOi www.oioi.fi

Game Design

  • Co-Writing Game Design Document.
  • Creating Asset list.
  • Maintaining Github project.
  • Game testing on-site.

Level Design

  • Asset management in Unity.
  • Co-Creating Level layouts based on delivered concept art.
  • Creating Lighting for scene in Unity URP.
  • Grayboxing and adding Final assets.

Working with a big touch wall meant taking into account players height and reach space. Illustrating it using rule of thirds gave a better understanding of collectable placements in the game.
